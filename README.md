# 🚢 Titanic Survival Prediction using Machine Learning

## 📌 Project Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning. It includes data preprocessing, feature engineering, exploratory data analysis (EDA), model training, evaluation, and business insights.

---

## 📂 Dataset

- Source: Kaggle
- Dataset: Titanic Dataset

---

## 🎯 Objectives

- Clean and preprocess the dataset
- Handle missing values
- Perform feature engineering
- Analyze passenger survival patterns
- Train a Logistic Regression model
- Evaluate model performance

---
